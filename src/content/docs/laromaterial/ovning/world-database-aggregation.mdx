I denna övningen fortsätter vi att använda databasen World. Nu skall vi titta på aggregerande funktioner och vyer.

Allt arbete vi gör sparar vi i en SQL-fil som vi kan återanvända senare.



## Förutsättningar

Du har jobbat igenom övningen "Övning databasen World: JOIN och rapporter".

Du har läst på om SQL.



## Spara ditt arbete i ditt kursrepo

I ditt kursrepo, skapa följande katalog där du kan spara ditt arbete med denna övning:

```bash title="Skapa katalog för övningen"
# Gå till ditt kursrepo
mkdir -p kmom/02/world2
cd kmom/02/world2

# Öppna katalogen i VS Code
code .
```

Skapa filen `aggregate.sql` och spara all SQL-kod du skriver på egen hand under övningen i den filen.

Börja med att lägga till följande header i filen `aggregate.sql`.

```sql title="aggregate.sql"
-- SQL-kod för övningen Databasen World
-- Kurs: Databasteknik
-- Författare: Ditt Namn
-- Datum: ÅÅÅÅ-MM-DD
```

Uppdatera med rätt datum och namn. Du kan skriva kommentarerna på engelska om du vill.
