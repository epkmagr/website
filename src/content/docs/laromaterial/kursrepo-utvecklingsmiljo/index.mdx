## Förutsättning

För att genomföra denna guiden behöver du tillgång till ditt kursrepo som du skapar. I denna guiden kommer jag att använda kursrepot `webtec-abcd25` som exempel.

Innan du börjar jobba i guiden så öppnar du en terminal och flyttar du till katalogen för ditt kursrepo.



## Guidens innehåll

I guiden får du hjälp med:

1. Lägg till konfigurationsfilen `.editorconfig` för inställningar i vscode.

1. Använd git-kommandon som add, commit, push, pull, status och log för att jobba med ditt git repo.

1. Lägg till filen `.gitignore` för att ignorera vissa filer och kataloger från att inkluderas i ditt repo.

1. Initiera ditt repo som ett npm projekt med `npm init`.

1. Installera lintern ESLint som hjälper dig att formattera och "linta" din kod.

1. Installera webbservern http-server som ett utvecklingsverktyg.
