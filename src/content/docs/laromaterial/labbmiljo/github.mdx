Webbverktyget GitHub används för att hantera dina Git repon. Du behöver skaffa ett konto på GitHub.

Du behöver också koppla ditt konto så att BTH vet vilket konto du vill använda för dina studier.



## GitHub

Webbverktyget [GitHub](https://github.com/) är ett verktyg som hjälper dig att publicera ett git repo så att andra kan komma åt det.

I kurserna används GitHub och du behöver därför skapa ett eget konto på GitHub. Det är kostnadsfritt. Ta hjälp av informationen på webbplatsen och skapa ett konto och logga in på webbplatsen.



## Koppla ditt konto på GitHub till BTH

När du är klar med ditt konto går du till Canvas och gör uppgiften "Github invite". Där skriver du enbart den epost du registrerade ditt konto med på GitHub och lämnar in.

Därefter får du en inbjudan till den eposten som du följer.

{/* När du är klar med ditt konto så besöker du följande sida på BTH's webbplats.

* "[Koppla ditt GitHub konto till BTH](https://bth.se/XXX)" (LÄNK BRUTEN, INVÄNTAR RIKTIG LÄNK) */}

Det material du skapar i kursen kommer att placeras på en organisation på GitHub som ägs av BTH och ditt material kommer att vara privat för dig och dina lärare.
