Kursen Databasteknik, a.k.a. databas, lär ut teori och praktik i databasteknik med relationsdatabaser och SQL tillsammans med applikationsprogrammering mot en databas.

Vi börjar med att installera en databas och komma igång med grunderna hur man jobbar mot en databas från terminalen och från desktop-verktyg.

Därefter börjar vi modellera olika databaser och vi skapar dem med SQL. Databaserna fylls med data och vi använder frågespråket SQL för att ställa frågor mot datat.

Näst steg blir att koppla en applikation mot databasen och där anvädner vi ett programmeringsspråk för att bygga applikationer som jobbar mot databasen.

Vi fortsätter med att lära oss mer om relationsdatabaser och hur vi kan jobba mot dem.



Innehåll
------------------------

Kursen omfattar följande områden:

* Databasteknik med fokus på relationsmodellen.

* Modellering av databaser med ER-modellering (Entity-Relationship).

* SQL för att skapa, uppdatera, radera och fråga databasen samt sammanställa rapporter.

* SQL med lagrade procedurer, funktioner och triggers.

* Transaktioner i databas.

* Databasprestanda och index.

* Programmering av terminalbaserade applikationer för databaser, med objektorienterade programmeringstekniker.

* Verktyg och tekniker som används vid utveckling av databaser och applikationer mot databaser.
